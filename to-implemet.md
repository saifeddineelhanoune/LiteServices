### Endpoints to Implement:

## POST /users: 
    Create a new user. You'll learn about reading the JSON request body and unmarshalling it into a User struct.

## GET /users:
    Get a list of all users.

## GET /users/:id: 
    Get a specific user by ID. You'll learn about route parameters (:id).

## PUT /users/:id: 
    Update a user's information.

## DELETE /users/:id: 
    Delete a user.
